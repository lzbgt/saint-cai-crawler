# Saint Cai Crawler

Python/JavaScript crawler proof of concept for extracting reader-page text and figures with requests, BeautifulSoup, Playwright/browser sessions, jsdom/CryptoJS helper reuse, and deobfuscated site scripts.

## Repository notes

See `NOTES.md` for the walkthrough of the current crawler shape. Treat target-site behavior as changeable and verify authorization before running extraction against any source.
